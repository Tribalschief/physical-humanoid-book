# High-Fidelity Rendering in Unity

While Gazebo is great for physics, Unity offers superior visual fidelity and is often used for Human-Robot Interaction (HRI) studies.

## Unity Robotics Hub
Unity provides tools to communicate with ROS 2, allowing us to use Unity as the visualization layer while ROS 2 handles the logic.
