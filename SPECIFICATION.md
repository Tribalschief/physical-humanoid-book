# Project Specification: Physical AI & Humanoid Robotics Textbook

## Overview
A comprehensive interactive textbook designed to teach Physical AI and Humanoid Robotics. The project utilizes a modern stack to deliver content, assess user background, and provide an AI-powered teaching assistant.

## Tech Stack
- **Frontend/Content**: Docusaurus (React, TypeScript)
- **Deployment**: GitHub Pages / Vercel
- **Backend (RAG)**: FastAPI, OpenAI Agents SDK
- **Database**: Neon (Serverless Postgres), Qdrant (Vector DB)
- **Auth**: Better-Auth

## Core Features
1.  **Curriculum Content**:
    - **Module 1**: The Robotic Nervous System (ROS 2)
    - **Module 2**: The Digital Twin (Gazebo & Unity)
    - **Module 3**: The AI-Robot Brain (NVIDIA Isaac)
    - **Module 4**: Vision-Language-Action (VLA)
2.  **Interactive RAG Chatbot**:
    - Context-aware answers based on book content.
    - Floating chat interface or embedded widget.
3.  **User Personalization**:
    - **Onboarding**: Survey on software/hardware background.
    - **Auth**: Signup/Login functionality.
    - **Adaptability**: "Personalize Content" button to adjust complexity/examples.
4.  **Localization**:
    - "Translate to Urdu" button for accessibility.
